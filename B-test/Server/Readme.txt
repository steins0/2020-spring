Qt版本为Qt Creator 5.7.0
注：本人为新手一个，程序由本人自主学习及网络参考编写，用于学习Qt编程；
   其中涉及很多知识点包括有Qt网络编程TCP、Qt多线程编程、Qt标准控件的使用等等；
   仅供需要的人学习参考，希望共同学习进步。

   此程序为局域网文件传输服务器端，支持多客户端的连接请求。

应用功能描述及注意事项：
    支持多客户端的请求连接，客户端连接上后会显示客户端的IP地址（最多显示三个客户端IP）；
    可主动断开客户端的连接；
    可以传输文件到客户端（当有多个客户端同时连接时，只能向所有客户端同时发送同一个文件）；
    支持添加文件夹目录到浏览区域，并可以双击进行目录的切换，可以通过删除条目进行删除。
    支持客户端的文件下载功能，允许被客户端下载的文件需要放置在ServerFile目录下；
    接收到的客户端传输过来的文件会保存在ReceiveFile目录下；
